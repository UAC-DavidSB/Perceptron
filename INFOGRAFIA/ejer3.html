🧠 Concepto Fundamental

El perceptrón representa la unidad computacional más elemental en redes neuronales artificiales. Con función de activación lineal, mantiene una relación directa entre entrada y salida, generando valores continuos sin transformación no-lineal.

⚖️ Análisis Comparativo

Ventajas

  • Salida continua y diferenciable en todo el dominio
  • Implementación computacionalmente eficiente
  • Ausencia de saturación de gradiente
  • Óptima para problemas de regresión lineal
  • Convergencia rápida en espacios linealmente separables

Limitaciones

  • Alta sensibilidad a la escala y distribución de datos
  • Salida no acotada: rango (-∞, +∞)
  • Incapacidad para modelar relaciones no-lineales
  • Performance subóptimo en clasificación compleja
  • Requiere normalización rigurosa de features

∇ Regla de Actualización

wᵢ ← wᵢ + η · error · xᵢ + ε
bias ← bias + η · error + ε

Donde η es el learning rate, error = y_true - y_pred, y ε es ruido gaussiano [-0.04, 0.04] para evitar mínimos locales.

💡 Insight Principal

La función de activación lineal produce outputs continuos sin límites, haciéndola inherentemente más adecuada para problemas de regresión que para clasificación. Su extrema sensibilidad a la escala de los datos requiere preprocesamiento meticuloso mediante normalización o estandarización para garantizar convergencia estable en la detección de spam.
