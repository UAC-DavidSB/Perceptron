🎓 Proceso de Entrenamiento

Inicialización

Pesos aleatorios (w₁, w₂) y bias (b) entre -1 y 1

Predicción

Suma ponderada de entradas:

z = w₁·x₁ + w₂·x₂ + b

Aplicar activación: y_pred = σ(z)

Cálculo del Error

error = y_real - y_pred

Actualización de Pesos

wᵢ = wᵢ + η · error · xᵢ
b = b + η · error

η (tasa de aprendizaje) = 0.1

Repetir

20 épocas sobre todos los datos

⚙️ Configuración

Parámetros del Modelo
  • Entradas: 2 (x₁, x₂)
  • Tasa de aprendizaje: 0.1
  • Activación: Sigmoid
  • Épocas: 20
  • Objetivo: OR lógico

🎯 Resultado

Después de 20 épocas, el perceptrón clasifica correctamente las 4 combinaciones de OR. Los pesos convergen hasta minimizar el error total.

💡 Nota Importante

El perceptrón simple resuelve problemas linealmente separables (OR, AND, NOT) pero NO puede resolver XOR.
